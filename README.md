# 🏢 Campus Employee Management System

A desktop-based Employee Management System developed using **Python, Tkinter, and SQLite**. The application helps organizations manage employee information, attendance, salary records, and reports through an intuitive graphical user interface.

---

## 📌 Features

### 🔐 Authentication
- Admin Login System
- Secure Login Interface

### 👨 Employee Management
- Add Employee
- Update Employee
- Delete Employee
- Search Employee
- View Employee Records

### 📅 Attendance Management
- Record Employee Attendance
- Update Attendance
- Delete Attendance
- View Attendance Records

### 💰 Salary Management
- Add Salary Details
- Calculate Net Salary
- View Salary Records

### 📊 Reports
- Employee Statistics
- Attendance Statistics
- Salary Statistics

### 📄 Export
- Export Employee Records to Excel (.xlsx)
- Export Employee Records to PDF (.pdf)

---

## 🚀 Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/EmployeeManagementSystem.git
```

### 2️⃣ Move into the project folder

```bash
cd EmployeeManagementSystem
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
python app.py
```
